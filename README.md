# Pixel Flow

A casual, one-tap mobile puzzle game. Tap colored pigs onto a looping conveyor belt; they automatically shoot matching pixel cubes. Manage belt capacity and 5 waiting slots to clear the picture.

## Status

Design phase. The repo currently contains only the design and implementation brief — no game code yet.

- **[`GDD.md`](./GDD.md)** — canonical Game Design Document (core loop, entities, mechanics, level format, UI, win/fail states, sample level).
- **[`PROMPT.md`](./PROMPT.md)** — self-contained engineering handoff prompt for Phaser 3 + TypeScript + Vite implementation, with five phased milestones and acceptance criteria.

## Getting Started (for implementers)

1. Read `GDD.md` end-to-end.
2. Read `PROMPT.md`.
3. Begin Phase 1 on a new branch (see `PROMPT.md` §4 and §8).

## Tech Stack (planned)

- Phaser 3 (latest stable)
- TypeScript (strict)
- Vite

See `PROMPT.md` §2–3 for the full constraint set and target repo layout.
